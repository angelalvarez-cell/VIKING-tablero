# VIKING-tablero

Tablero de producción del taller Viking (GAV Detailing).

- **Taller** — pantalla para la TV (output, solo lectura).
- **Tableta** — input de técnicos (avanzar hitos + carril Kevlar).
- **Control** — panel de administración (clave requerida).

Backend: Google Apps Script + Google Sheets (archivo `Code.gs`, se instala aparte en Apps Script).
Frontend: React + Vite, desplegado en Vercel.
